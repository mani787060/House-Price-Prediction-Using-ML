# House Price Prediction Using Machine Learning

This project builds a **regression model** to predict housing prices based on multiple property features such as area, rooms, age, location, and more.  
It demonstrates a full ML workflow including data preprocessing, feature engineering, model training, and evaluation.

## Contents
- Exploratory Data Analysis (EDA)
- Handling missing values and outliers
- Feature scaling & encoding
- Train-test split
- Model training (Linear Regression / Ridge / Lasso / Random Forest)
- Performance evaluation (MAE, MSE, RMSE, RÂ² Score)
- Predicting house prices for new input data

## Key Concepts
- Regression modeling for real estate
- Importance of feature engineering
- Comparing multiple ML algorithms
- Visualizing prediction errors
